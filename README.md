# StructuralDynamicsODESolvers.jl

This package is a (just starting) Julia implementation of common ODE solvers for
structural dynamics problems.

## Features

- Solvers for linear dynamic equations `Mx''(t) + Cx'(t) + Kx(t) = F(t)`

    - Central difference
    - Houbolt
    - Newmark
    - Bathe
